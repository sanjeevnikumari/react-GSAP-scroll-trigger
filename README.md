
# ✨ React GSAP ScrollTrigger Image Reveal

A modern scroll-based image reveal animation built using **React, GSAP, and ScrollTrigger**.  
Images animate from outside the viewport and smoothly arrange into a dynamic layout as the user scrolls.

This project demonstrates advanced scroll animations, timeline control, and high-performance rendering using GSAP with React.

## ✨ Features

- Smooth scroll-based animation  
- GSAP ScrollTrigger integration  
- Pinned scroll section  
- Staggered image animation  
- Random initial rotation for realistic motion  
- Fully responsive design  
- High performance animation
- Clean and modern UI
